# 🎓 Canara Infomate Bot


An intelligent virtual assistant for **Canara Engineering College** that streamlines access to academic, admission, placement, and campus information through a conversational interface.

The chatbot leverages  **Google Search Grounding** to provide accurate, context-aware responses based on the official college website.

---

## 🚀 Features

- 🤖 Intelligent conversational chatbot
- 🎓 Admission and eligibility information
- 📚 Academic and department details
- 🏫 Campus facilities and student services
- 🔍 Google Search Grounding for reliable responses
- 📱 Responsive design for desktop and mobile
- ⚡ Fast and intuitive user experience

## 💡 How It Works

1. User enters a question.
2. The chatbot sends the query .
3. Google Search Grounding retrieves relevant information from the official college website.
4. It generates a concise and contextual response.
5. The chatbot displays the response along with relevant source links when available.
